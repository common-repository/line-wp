== Description ==

You can link articles posted on WordPress to LINE.
We will add a function to link LINE and WordPress.

= Key Features =
Articles posted on WordPress are automatically posted to LINE for Business messages.
You can also post an eye-catching image.
The message (explanation) at the time of posting can be your own text instead of a summary.

== Installation ==

= Minimum Requirements =

* WordPress 5.7 or greater
* PHP version 7.2 or greater
* MySQL version 5.6 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of LINE for WordPress, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “LINE for WordPress” and click Search Plugins. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =
The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application.
